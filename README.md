# GYAN-GraphQL

A GraphQL service for football Data.

## Requirements

* [NodeJS](http://nodejs.org) 4.8 or above

## Install

```sh
npm install
```
or `yarn install` as alternative

## Build

`npm run build` or `yarn build`

## Test

* **Unit** - `npm test`
start
npm start or sudo -E  docker-compose up --build
